☂️ ToolKitV (Enhanced Fork)
ToolKitV Enhanced é uma versão otimizada e expandida do utilitário original da Umbrella.re. Este fork foi desenvolvido para criadores que precisam de máxima eficiência, permitindo a otimização em massa de texturas e gestão avançada de assets para FiveM/GTA V.

🚀 O que há de novo neste Fork?
Processamento em Paralelo: Otimização de texturas em simultâneo, aproveitando todos os núcleos do seu CPU para reduzir o tempo de espera em até 80%.

Engine "Lean": Remoção de verificações redundantes e chamadas de API desnecessárias, tornando a execução muito mais leve e rápida.

Vehicle Pack Compressor: Nova funcionalidade para comprimir e agrupar múltiplos veículos num único pack otimizado automaticamente.

Auto-fix de Mips: Geração automática de mipmaps para evitar o efeito "flicker" em texturas de baixa resolução.

✨ Funcionalidades Principais
Otimização de Texturas: Conversão e compressão inteligente com suporte a múltiplos formatos.

Multi-threading: Processa centenas de texturas de uma só vez.

Vehicle Tools: Ferramenta dedicada para compressão de veículos (YTDs e YFTs).

Interface Simplificada: Menos janelas, mais ação.

🛠 Requisitos
Ambiente: .NET 8.0 Runtime (ou superior) para melhor performance de memória.

Sistema: Windows 10/11 (64-bit).

📥 Instalação & Uso
Faz o download da última Release.

Extrai os ficheiros para uma pasta à tua escolha.

Executa o ToolKitV.exe.

Para o compressor de carros, basta arrastar a pasta dos teus veículos para a aba "Vehicle Compressor".
